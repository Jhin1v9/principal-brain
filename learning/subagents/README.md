# Subagent Learning Memory

## Purpose

Store reusable lessons from delegated missions so the principal agent improves over time.

## What belongs here

- mission summaries
- delegation mistakes
- role effectiveness notes
- protocol improvements
- failure patterns
- reusable team recipes

## Rule

The latest verified note should become the operational truth, but history should remain visible.
