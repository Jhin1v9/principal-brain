## Lições
- Evitar importar bibliotecas CSS inteiras para poucos ícones — copiar assets estáticos é mais eficiente
- Sempre verificar `publicDir` em todos os apps do monorepo quando adicionar assets públicos
- IDs do Unsplash não são confiáveis para download programático — verificar visualmente cada imagem
